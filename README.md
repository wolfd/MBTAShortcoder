MBTAShortcoder
==============

WordPress shortcode plugin that checks subway times for a particular stop and line combination.

Examples:
```
[mbtastatus line="orange" stop="Ruggles"]
[mbtastatus line="red" stop="Park Street"]
```
In the post editor
